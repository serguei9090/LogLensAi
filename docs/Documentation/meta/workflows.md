# AI Workflows & Commands: LogLensAi

LogLensAi includes a suite of specialized AI workflows and slash commands designed to automate complex development and maintenance tasks.

## 🛠️ Implementation Workflows

| Command | Workflow File | Description |
|---|---|---|
| `/startcycle` | `startcycle.md` | The main autonomous development pipeline. Orchestrates planning and execution. |
| `/feature` | `feature.md` | Used for implementing new feature requests with formal specs. |
| `/fixcycle` | `fixcycle.md` | Systematic issue identification and resolution loop. |
| `/jules-cycle` | `jules-cycle.md` | Automated "Vibe-to-Code" cycle for rapid prototyping. |

## 🧪 Quality & Testing Workflows

| Command | Workflow File | Description |
|---|---|---|
| `/autolint` | `autolint.md` | Runs Biome and Ruff in a loop to ensure 100% compliance. |
| `/testcov` | `testcov.md` | Identifies coverage gaps and generates tests to fill them. |
| `/quality-report` | `quality-report.md` | Generates a technical debt and code quality audit report. |
